# tunistrade-pages
Static website files for TunisTrade's front-end, including HTML, CSS, and JavaScript for the shop and user interface.



## Project Structure

```bash
/tunistrade-pages
├── /static                      # Static assets for the frontend
│      ├── /css                  # CSS files for styles
│      │      ├── bestellungen.css  # Styles for the Bestellungen page
│      │      ├── b2b.css           # Styles for the B2B page
│      │      ├── crm.css           # Styles for the CRM page
│      │      ├── index.css         # Styles for the index (home) page
│      │      ├── lager.css         # Styles for the Lager (warehouse) page
│      │      ├── login.css         # Styles for the login page
│      │      ├── packen.css        # Styles for the Packen page
│      │      ├── style.css         # General shared styles
│      │      ├── onlineshop.css    # Styles for the onlineshop page
│      │      ├── umsatz.css        # Styles for the Umsatz page
│      │      ├── versendet.css     # Styles for the Versendet page
│      ├── /img                  # Directory for image assets
│      ├── /js                   # JavaScript files for functionality
│      │      ├── bestellungen.js   # JavaScript for the Bestellungen page
│      │      ├── b2b.js            # JavaScript for the B2B page
│      │      ├── crm.js            # JavaScript for the CRM page
│      │      ├── index.js          # JavaScript for the index (home) page
│      │      ├── lager.js          # JavaScript for the Lager (warehouse) page
│      │      ├── login.js          # JavaScript for the login page
│      │      ├── packen.js         # JavaScript for the Packen page
│      │      ├── onlineshop.js     # JavaScript for the onlineshop page
│      │      ├── umsatz.js         # JavaScript for the Umsatz page
│      │      ├── versendet.js      # JavaScript for the Versendet page
├── bestellungen.html            # HTML for the Bestellungen page
├── b2b.html                     # HTML for the B2B page
├── crm.html                     # HTML for the CRM page
├── index.html                   # HTML for the index (home) page
├── lager.html                   # HTML for the Lager (warehouse) page
├── login.html                   # HTML for the login page
├── packen.html                  # HTML for the Packen page
├── onlineshop.html              # HTML for the shop page
├── umsatz.html                  # HTML for the Umsatz page
├── versendet.html               # HTML for the Versendet page
```
